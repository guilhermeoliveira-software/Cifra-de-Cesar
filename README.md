🔐 Cifra de César em Java
Este projeto foi desenvolvido como parte da disciplina de Cyber Security na UniFametro. O objetivo principal foi implementar um dos algoritmos de criptografia mais antigos e conhecidos da história: a Cifra de César.

📖 Sobre o Projeto
A Cifra de César é um tipo de criptografia por substituição no qual cada letra do texto original é substituída por outra letra que se apresenta no alfabeto abaixo dela um número fixo de vezes (o deslocamento ou chave).

O foco deste desenvolvimento foi aplicar lógica de programação em Java para manipulação de caracteres e strings, garantindo a integridade da mensagem criptografada.

🚀 Funcionalidades
Criptografia: Transforma um texto claro em texto cifrado com base numa chave numérica.

Descriptografia: Reverte o processo, recuperando a mensagem original utilizando a mesma chave.

Tratamento de Caracteres: Mantém a formatação original de espaços e caracteres especiais.
